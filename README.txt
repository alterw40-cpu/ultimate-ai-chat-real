Ultimate AI Chat - Real - How to run

1. Install Node.js LTS from https://nodejs.org (if not done already).

2. Open Command Prompt and navigate to this folder:
   cd path\to\ultimate-ai-chat-real

3. Install dependencies:
   npm install

4. Run the app locally:
   npm run dev

5. Open your browser and go to:
   http://localhost:3000

6. Wait a few seconds while the AI model downloads and loads.

7. Start chatting with the AI running locally!

8. To stop the server, press Ctrl+C in Command Prompt.

---

To deploy online:

- Push this folder to a GitHub repo.

- Connect your repo to Vercel or Netlify (free plans).

- Deploy and share the live URL with buyers.

---

Rebranding:

- Change “Ultimate AI Chat” text in `src/App.jsx` inside `<h1>` tag.

- Replace styles or add your own logo as needed.
